# Interactive Groceries

אפליקציית SwiftUI מותאמת לאייפון לניהול רשימות קניות אינטראקטיביות ומודרניות.

## יכולות עיקריות
- הוספת פריטים עם קטגוריות, הערות, עדיפות וכמות.
- ניהול רשימה דינמי עם חיפוש, מיון וסינון בזמן אמת.
- מעקב אחרי היסטוריה והצגת תובנות וסטטיסטיקות חכמות.
- התאמת ערכת צבעים וממשק משתמש מודרני עם אנימציות.
- סנכרון נתונים מקומי ושיתוף קל של הרשימה דרך העתקה ל-JSON.

## הוראות הפעלה
1. פתחו את Xcode וצרו פרויקט חדש מסוג **App** עם SwiftUI.
2. העתיקו את כל תיקיית `InteractiveGroceries` אל תיקיית המקור של האפליקציה.
3. ודאו שהיעד הוא iOS 17 ומעלה כדי ליהנות מכל אפקטי החזותיים והאנימציות.
4. בחרו ב-`InteractiveGroceriesApp` כנקודת הכניסה (`@main`).

## הרחבות אפשריות
- שילוב iCloud Sync כדי לשתף רשימות בין מכשירים.
- הוספת תזכורות מבוססות מיקום ו-Notifications כאשר חסרים פריטים קריטיים.
- בניית ווידג'טים ל-iOS להצגת הרשימה במסך הבית.
